# CampusCompass

Student benefits finder — built step by step as a portfolio project.

## Current phase: 0 (landing page only)

Right now this repo is intentionally minimal:

- Next.js + TypeScript + Tailwind
- One landing page at `/`
- No quiz, API, database, or AI yet — those come in later phases

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project structure (Phase 0)

```
src/app/
  layout.tsx   # Wraps every page (fonts, HTML shell)
  page.tsx     # The homepage (/)
  globals.css  # Tailwind + global styles
```
